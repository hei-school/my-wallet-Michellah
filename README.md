# Java Installation Guide

This guide provides step-by-step instructions to install Java on your operating system. Java is a widely used technology for application development.

## Prerequisites

Make sure you have the following before starting the installation:

- Access to an account with administrative privileges (if necessary).
- Active internet connection to download the installation files.

## Installation Steps

Follow these steps to install Java on your system:

### Step 1: Download Java

1. Go to the official Java website at [java.com](https://www.java.com/).
2. Click the download button to get the latest version of Java.

### Step 2: Installation on Windows

1. Once the installation file is downloaded, double-click it to start the installation process.
2. Follow the on-screen instructions. Accept the terms of the license agreement if prompted.
3. Choose the installation location if prompted.
4. Wait for the installation to complete.

### Step 3: Installation on macOS

1. Open the downloaded installation file (.dmg).
2. Double-click the installation package.
3. Follow the on-screen instructions to complete the installation.

### Step 4: Verification of Installation

After the installation, check if Java has been installed correctly by opening a terminal or command prompt and typing the following command:

    ```bash
    java -version
